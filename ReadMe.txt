1) Loading_Wine_Dataset.py

	It loads the Wine Recognition dataset, preprocesses it, reduce outliers and visualisation using heatmap.

----------------------------------------------------------------------------------------------------------
2) Clustering.py

	Clustering.py file contains 5 clustering methods 
	(K-means, Hierarchical Agglomerative, GausianMixtureModel, MeanShift Clustering, Birch Clustering)

----------------------------------------------------------------------------------------------------------
3) Classification.py

	Classification.py  file contains 5 classification methods 
	(DecisionTreeClassifier, SupportVectorClassifier, RandomForestClassifier, Logistic regression, K-Neighbors Classifier)

The below link is for downloading the Scikit library.
We are using Scikit-sklearn library here (https://scikit-learn.org/stable/)
